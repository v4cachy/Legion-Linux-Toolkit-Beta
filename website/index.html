What it offers

Everything you need, nothing you don't

Full hardware control from a single dashboard. No background bloat, no Windows required.

⚡

Power Profiles

Switch between Quiet, Balanced, Performance and Custom profiles. Also cycle with Fn+Q. Auto-switch on AC connect or battery.

🌈

RGB Keyboard

4-zone RGB control via LegionAura — Static, Breath, Wave, Hue and Off effects. Per-zone colour pickers with quick presets.

🎡

Fan Monitoring

Live animated fan icons with real-time RPM. Auto or Full Speed modes. Fan curves baked into each power profile.

🖥️

Display Control

Screen brightness slider. Independent resolution and refresh rate selectors. Display Overdrive and G-Sync toggles.

🔋

Battery Care

Conservation mode (~60% cap), Rapid Charge, USB Charging. ThinkPad charge threshold controls with custom start/stop %.

🎮

GPU Mode Switching

Switch between Hybrid, NVIDIA-only and Integrated GPU modes via envycontrol — directly from the home screen.

🚀

Overclock

CPU max/min frequency sliders, TDP PL1/PL2 controls and GPU core/memory offset with a master enable toggle for safety.

🌍

11 Languages

First-run wizard lets you choose your language. English, Français, Deutsch, Español, Türkçe, Русский, 中文, 日本語 and more.

🔍

Hardware Detection

One-time hardware scan on first launch. Detects your Lenovo brand and shows only the features your device actually supports.

Get started

Install in seconds

One script handles everything — dependencies, hardware detection, daemon service and tray launch.

1

Clone the repo

Download the latest code from GitHub to your machine.

2

Run the installer

The installer auto-detects your Lenovo brand and installs only what your device needs.

3

Setup wizard

On first launch, choose your language and let the hardware scan run once. Done.

4

Stay updated

Run update.sh anytime to pull the latest changes from GitHub and restart everything automatically.

Install
# Clone and install git clone https://github.com/v4cachy/legion-linux-toolkit cd legion-linux-toolkit sudo bash install.sh
Update
# Pull latest from GitHub cd ~/legion-toolkit sudo bash update.sh
Requirements
# Auto-installed by install.sh python-pyqt6 qt6-wayland libnotify kscreen # Optional (auto by brand) lenovolegionlinux envycontrol legionaura
